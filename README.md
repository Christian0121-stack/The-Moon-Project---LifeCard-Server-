🚀 LifeCard Server (Admin Central Command)
The LifeCard Server is the central management hub for the LifeCard ecosystem. It acts as the "Administrative Command Center," ensuring secure data provisioning, hardware fulfillment, and system-wide communication.

🛠 Core Functionalities
1. User Provisioning & Management
To maintain medical integrity, LifeCard uses a Closed-Loop System.

Administrative Control: Create, update, or deactivate Admin, Doctor, and Patient accounts.

Strict Access: Users cannot self-register; all credentials are provisioned here to prevent unauthorized access to sensitive medical records.

2. Fulfillment & Logistics (Orders)
Manage the distribution of the physical LifeCard ecosystem:

Physical Cards: Track and process orders for standard LifeCard IDs.

LifeCard+ (RFID/NFC): Organize and fulfill orders for wearable RFID/NFC bracelets.

3. Broadcast Center (Announcements)
Deploy critical updates or system-wide news.

Announcements are instantly synced and displayed on both the Doctor’s Interface and the Patient App.
